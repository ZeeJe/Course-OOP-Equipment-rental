#  Система управління прокатом інвентарю (Equipment Rental)

![.NET CI/CD Pipeline](https://github.com/ZeeJe/Course-OOP-Equipment-rental/actions/workflows/dotnet.yml/badge.svg)

##  Опис проєкту
Цей проєкт є інтелектуальною системою для автоматизації процесів прокату легкого транспорту (електросамокатів та велосипедів). Розроблений як курсова робота з дисципліни "Об'єктно-орієнтоване програмування".

Система забезпечує повний цикл взаємодії з користувачем: від реєстрації та вибору техніки на інтерактивній карті до автоматичного розрахунку вартості поїздки на основі різних тарифних планів та списання коштів.

##  Технологічний стек
* **Мова:** C# / .NET 10
* **Інтерфейс:** ASP.NET Core Blazor (Interactive Server)
* **Архітектура:** MVC (Model-View-Controller) з дотриманням принципів SOLID
* **База даних:** MySQL + Entity Framework Core
* **Тестування:** xUnit + Moq (In-Memory DB для ізоляції тестів)
* **DevOps:** CI/CD через GitHub Actions

---

##  Реалізовані патерни
Проєкт демонструє практичне застосування класичних патернів GoF:
* **Strategy:** Динамічний вибір алгоритму розрахунку ціни (Fixed, Per-Minute, Subscription).
* **State:** Керування життєвим циклом транспорту (Available, InUse, Maintenance).
* **Factory Method:** Ізольоване створення екземплярів різних типів транспорту.

---

##  Тестування та Якість (Quality Assurance)
Система має **100% покриття критичної логіки** (загалом **51 юніт-тест**). 
Автоматизоване тестування виконується при кожному `push` у репозиторій за допомогою **GitHub Actions**, що гарантує стабільність коду.

Перевірено блоки:
- Розрахунок усіх типів тарифів та промокодів.
- Коректність зміни станів транспорту.
- Логіка списання коштів та перевірка балансу користувача.
- Робота репозиторіїв та обробка кастомних винятків (Exceptions).

##  Як запустити проєкт
1. **Клонування:**
    ```bash
    git clone [https://github.com/ZeeJe/Course-OOP-Equipment-rental.git](https://github.com/ZeeJe/Course-OOP-Equipment-rental.git)
    ```
2. **Налаштування БД:** Оновіть `ConnectionString` у файлі `appsettings.json`.
3. **Запуск:**
    ```bash
    dotnet run --project BlazorApp
    ```
4. **Запуск тестів:**
    ```bash
    dotnet test
    ```
